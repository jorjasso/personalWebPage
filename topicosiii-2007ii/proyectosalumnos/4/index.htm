Algoritmo:

 

El algoritmo utilizado para la implementación de las TDC I  IV y la TDC Modificada está basado en las formas de las transformadas, por lo que tiene una complejidad de O(n2).

Conclusiones

 

  • La complejidad del algoritmo utilizado para implementar las TDCs puede ser mejorado utilizando alguna de las técnicas que existen para la implementación, como por ejemplo el algoritmo radix  2, que debe ser adaptado para su aplicación a las TDC.

 

  • Las diferencias entre las señales obtenidas luego de aplicar las distintas TDCs son mínimas y casi imperceptibles como se puede observar en las imágenes.

 

  • Así como las diferencias entre las señales obtenidas no son muy marcadas, también lo son entre los espectrogramas de las TDCs, en las cuales se aplica un filtro pre-énfasis de -0.95.

 

  • En la TDC Modificada se puede apreciar una mayor diferencia con las otras. Esto debido a que la TDC Modificada hace uso sólo de la mitad de la señal que le es ingresada y como consecuencia su espectrograma correspondiente tiene menor resolución que las otras.
